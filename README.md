# ------ scale practice game ------- 
## purpose: i suck at memorizing scales so i want to make an app to help me practice 
## overview: read auto input through audio interface(good luck) and play a buzzer soundwhenever a note outside of the scale is played 
## features: 
 - select scale 
 - live input reading to identify scale degrees 
 - metronome with bpm selection
 - timer/timer-based score system based on correct/wrong notes
 - database to keep track of past scored
 - graph vizualization of progress 
## mvp: 
 - scale selections 
 - live input reading
